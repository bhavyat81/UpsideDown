# Assets

Place the following asset files here before publishing the app:

| File | Size | Description |
|---|---|---|
| `icon.png` | 1024×1024 | App icon |
| `splash.png` | 1284×2778 | Splash screen image |
| `adaptive-icon.png` | 1024×1024 | Android adaptive icon foreground |
| `favicon.png` | 48×48 | Web favicon |

You can generate these assets using the [Expo icon guide](https://docs.expo.dev/develop/user-interface/splash-screen-and-app-icon/).
